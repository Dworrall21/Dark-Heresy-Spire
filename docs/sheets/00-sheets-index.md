# Sheets Index v0.1

This folder contains table-facing sheets for **Dark Heresy Spire**, a Spire-style Inquisition Resistance hack.

These sheets are intended to work first as Markdown documents and later as standalone site pages.

## Sheets

1. [Acolyte Character Sheet](01-acolyte-character-sheet-v0.1.md)
2. [Cell Sheet](02-cell-sheet-v0.1.md)
3. [Patron Sheet](03-patron-sheet-v0.1.md)
4. [Operation Sheet](04-operation-sheet-v0.1.md)
5. [Quick Reference Sheet](05-quick-reference-sheet-v0.1.md)

## Recommended Use

Each player should have an **Acolyte Character Sheet**.

The table should share:

- One **Cell Sheet**
- One **Patron Sheet**
- One **Operation Sheet** per active Operation
- One **Quick Reference Sheet** for the table

## Future Site Page Goals

Later, each sheet can become a dedicated site page with:

- Fillable text fields
- Clickable stress tracks
- Expandable help text
- Printable layout
- Mobile-friendly reference sections
- Links to relevant rules sections
