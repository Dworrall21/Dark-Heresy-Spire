2. The GM rolls the stress die.
3. Add the result to that Resistance.

# Bonds

Create three Bonds:

1. One Bond to another acolyte.
2. One Bond to someone outside the cell.
3. One Bond to a faction, place, secret, or former life.

| Bond | Die | Why They Matter | What They Provide | What Puts Them at Risk |
|---|---:|---|---|---|
| **Acolyte Bond** | D6 / D8 / D10 |  |  |  |
| **External Bond** | D6 / D8 / D10 |  |  |  |
| **Faction / Secret Bond** | D6 / D8 / D10 |  |  |  |

When a Bond is harmed, exhausted, compromised, or endangered, step its die down.

If it drops below D3, the Bond breaks, dies, betrays you, disappears, or becomes a threat.
