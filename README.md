# Dark Heresy Spire

**Dark Heresy Spire** is a fan design project adapting Spire / Resistance Toolbox-style play to an **Imperium Maledictum / Inquisition** campaign.

The working pitch:

> You are doomed agents of an oppressive empire trying to prevent something worse.

The game focuses on Inquisitorial acolytes: competent, dangerous, deniable agents serving a Patron who grants authority, resources, secrecy, and judgment.

This project is currently in a **v0.1 playtest chassis** state. The core rules, character creation, classes, Patron pressure, Heat/Subtlety, GM tools, requisition/gear, psychic and forbidden methods, table sheets, and a first playtest mission are present enough to run a one-shot or short test mission.

## Current Playtest Status

The current build supports:

- Core D10 pool rolls.
- Lex Ecclesiastica-style Risky/Dangerous difficulty, where the highest die or dice are removed after rolling.
- Five Resistances: Body, Mind, Shadow, Authority, Corruption.
- Minor, Moderate, Severe, and Critical Fallout.
- Beat-driven advancement.
- Complete playtest class packages with modular starting gear.
- Patron creation and Patron pressure.
- Heat/Subtlety and Enemy Awareness at the Operation level.
- Mission/Operation/Situation prep.
- GM-facing threat, clue, faction, clock, and NPC tools.
- Modular requisition and gear.
- Psychic powers, Warp Trace, and forbidden methods.
- A first playtest mission: **The Saint with the Wrong Shadow**.
- Player and table sheets.

## Design Notes

The project is built around a few assumptions:

1. Acolytes are competent; rolls are about cost, exposure, and consequence.
2. Investigation should move forward even when rolls go badly.
3. Authority should work, but never cleanly.
4. Gear is not a shopping list; gear is permission, pressure, leverage, and evidence.
5. Forbidden solutions should be tempting because they are effective.
6. Fallout should create more story instead of stopping play.
